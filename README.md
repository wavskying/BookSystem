# BookSystem
基于springboot+mybatis+layui+echarts的图书管理系统

## 拉去项目
git clone 

## 运行
修改application.yml文件中的数据库配置：sping:datasource下的url以及用户名密码

运行mysql脚本（内附有数千条数据）
